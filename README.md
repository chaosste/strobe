## About

Strobe Core turns your screen into a strobe light. That's it. That's the app.

AI-powered scene generation, precision frequency control, audio-reactive modes via your microphone — all the things you'd want from a strobe light that lives on your device. Stick it on a big telly at a house party. Point your phone at the ceiling. Use it however you like. We won't judge.

Maybe what AI was invented for.

## Features

- ⚡ **Precision Frequency Control** — Fine-tuned strobe rate, because detail matters
- 🤖 **AI Scene Generation** — Gemini-powered visual scenes (yes, really)
- 🎤 **Audio-Reactive Mode** — Strobes respond to microphone input in real time
- 🎨 **Colour Customisation** — Full control over strobe colour output
- 💡 **Multiple Strobe Styles** — Fixed frequency and dynamic patterns
- 🛡️ **Safety-First Design** — Mandatory disclaimer before any visual output

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + TypeScript, Vite |
| AI | Google Gemini API |
| Audio | Web Audio API (microphone input) |
| Deployment | Docker / Google Cloud |

## Installation

```bash
# Clone the repository
git clone https://github.com/chaosste/strobe.git
cd strobe

# Install dependencies
npm install

# Configure your Gemini API key

# Run development server
npm run dev
```
