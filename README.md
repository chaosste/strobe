## About

StrobeAI is a high-performance strobe light application with AI-powered scene generation. It delivers precision frequency control, audio-reactive modes via microphone input, and multiple strobe styles — all wrapped in a safety-first design that requires explicit acknowledgement before use.

## Key Features

- **Precision Frequency Control** — Fine-tuned strobe rate adjustment
- **AI Scene Generation** — Intelligent visual scenes powered by Gemini
- **Audio-Reactive Mode** — Real-time response to microphone input
- **Multiple Strobe Styles** — Fixed frequency and dynamic patterns
- **Color Customization** — Full control over strobe colour output
- **Safety-First Design** — Mandatory disclaimer before any visual output

## Tech Stack

- **Frontend:** React + TypeScript, Vite
- **AI:** Google Gemini API
- **Audio:** Web Audio API (microphone input)
- **Deployment:** Docker / Google Cloud

## Getting Started

```bash
# Clone the repository
git clone https://github.com/chaosste/strobe.git
cd strobe

# Install dependencies
npm install

# Set up environment
# Requires a Google Gemini API key

# Run development server
npm run dev
```
